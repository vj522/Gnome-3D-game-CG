# Gnome-3D-game-CG
Game for the Computer Graphics class.

### instructions
From the project's source code directory, run `python -m http.server 8000` in the terminal. Then open your web browser and launch the application by navigating to `http://localhost:8000`.


### short summary
This is a first-person game where you play as a forest gnome exploring a forest and a cave, collecting the correct items hidden throughout the environment. The items you need to collect are displayed on the screen. When you pick up a correct item, it glows and floats before you. If you collect the wrong item, you become poisoned, causing your vision to become blurry for a few seconds. You have 100 seconds to collect all the required items.
