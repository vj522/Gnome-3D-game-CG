# Gnome-3D-game-CG
game for computer graphics

## instructions
Iz lokacije mape koda v terminalu napišemo `python -m http.server 8000` ter v brskalniku igro zaženemo z `localhost:8000`



### short summary

First person game, smo gnome ki se sprehaja po gozdu in jami in rabi v pobirat prave elemente skrite v okolici. Ko pobere pravega se ta zasveti in lebdi, če pobere napačnega se zastrupi; ima meglen vid za nekaj sekund. Na voljo imamo 100 sekund.
