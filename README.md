# Gnome-3D-game-CG
game for computer graphics

## instructions
Iz lokacije mape koda v terminalu napišemo `python -m http.server 8000` ter v brskalniku igro zaženemo z `localhost:8000`



### short summary

First person game, smo gnome ki se sprehaja po gozdu ter jami in rabimo pobirat prave elemente skrite v okolici. Ko poberemo pravega se ta zasveti in lebdi, če poberemo napačnega se zastrupimo; imamo meglen vid za nekaj sekund. Na voljo imamo 100 sekund.
